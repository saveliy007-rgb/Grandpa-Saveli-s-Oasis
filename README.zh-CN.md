## 工作原理

1. **涡轮** 随风旋转 — 吸入空气。
2. **旋风分离器** 清除气流中的灰尘和沙子。
3. **管中管** 将空气输送到地下3-5米深处。
4. **不锈钢** 去除水分子的静电荷 → 蒸汽凝结成水滴。
5. **土壤（约+8°C）** 吸收多余的热量（相变）。
6. **土工布灯芯** 将蒸馏水提升到根部（像煤油灯一样）。

水是蒸馏水 — 一张白纸。  
您可以在灯芯中加入矿物质 — 树木将获得养分。

## 为何重要

每棵树都有自己的井。  
沙漠不必永远是沙漠。

## 安全性

可以增加一个由超级电容器供电的无线电信标。  
如果涡轮停止旋转，信标会发射信号：“需要维修”。
